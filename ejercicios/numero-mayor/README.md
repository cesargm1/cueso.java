## Arrays ejercicios

```java
public class App {
public static void main(String[] args) throws Exception {
int numeros[] = { 1, 4, 6, 7, 8, 9, 17, 3, 45, 6, 5, 0 };

        int sum = 0;

        for (int numero : numeros) {
            sum += numero;
        }

        System.out.println(sum);
    }

}
```
